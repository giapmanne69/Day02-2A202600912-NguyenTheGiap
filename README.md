# Hướng dẫn sử dụng và mô tả các file bài tập

Thư mục này bao gồm 3 file Markdown chính phục vụ cho quá trình làm bài tập cá nhân và nhóm:

## 1. 01-individual-problem-scan.md
- **Mục đích:** Ghi lại quá trình phân tích, nhận diện vấn đề cá nhân.
- **Nội dung:**
  - Mô tả vấn đề được giao.
  - Các bước phân tích, nhận diện vấn đề.
  - Kết quả phân tích cá nhân.

## 2. 02-group-problem-statement.md
- **Mục đích:** Tổng hợp, thống nhất vấn đề nhóm đã xác định.
- **Nội dung:**
  - Tóm tắt các ý kiến cá nhân.
  - Thảo luận và thống nhất vấn đề nhóm.
  - Đưa ra phát biểu vấn đề cuối cùng của nhóm.

## 3. 03-individual-reflection.md
- **Mục đích:** Ghi lại cảm nhận, đánh giá cá nhân sau khi hoàn thành bài tập.
- **Nội dung:**
  - Những gì đã học được qua bài tập.
  - Khó khăn gặp phải và cách giải quyết.
  - Đề xuất cải tiến cho các lần sau.

---

**Hướng dẫn sử dụng:**
- Mỗi thành viên hoàn thành file 01 và 03 theo ý kiến cá nhân.
- File 02 được hoàn thiện dựa trên thảo luận nhóm.
- Nên sử dụng định dạng Markdown để trình bày rõ ràng, dễ đọc.
